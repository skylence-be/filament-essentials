# Skylence Filament Essentials

Meta-package with essential Filament plugins. One install for the plugins you always need.

## Installation

```bash
composer require skylence/filament-essentials
```

## What's Included

### Runtime

| Package | What it does |
|---------|-------------|
| [filament/filament](https://github.com/filamentphp/filament) | Filament admin panel framework |
| [bezhansalleh/filament-shield](https://github.com/bezhanSalleh/filament-shield) | Roles and permissions for Filament |
| [dutchcodingcompany/filament-socialite](https://github.com/DutchCodingCompany/filament-socialite) | Social login for Filament |
| [shuvroroy/filament-spatie-laravel-backup](https://github.com/shuvroroy/filament-spatie-laravel-backup) | Backup management dashboard for Filament |
| [shuvroroy/filament-spatie-laravel-health](https://github.com/shuvroroy/filament-spatie-laravel-health) | Health check dashboard for Filament |
| [skylence/filament-context-sidebar](https://github.com/skylence-be/filament-context-sidebar) | Contextual sidebar for Filament panels |
| [skylence/filament-mega-menu](https://github.com/skylence-be/filament-mega-menu) | Mega menu navigation for Filament |
| [skylence/filament-system-configuration](https://github.com/skylence-be/filament-system-configuration) | System configuration UI for Filament |
| [skylence/filament5-glass-theme](https://github.com/skylence-be/filament5-glass-theme) | Glass theme for Filament 5 |
| [stephenjude/filament-feature-flags](https://github.com/stephenjude/filament-feature-flags) | Feature flags management for Filament |

### Dev (require-dev)

| Package | What it does |
|---------|-------------|
| [bezhansalleh/filament-plugin-essentials](https://github.com/bezhanSalleh/filament-plugin-essentials) | Essentials for Filament plugin development |
| [filament/blueprint](https://github.com/filamentphp/blueprint) | Generate Filament resources from models |
| [laraveldaily/filacheck-pro](https://laraveldaily.com/filacheck) | Filament code quality checker |
